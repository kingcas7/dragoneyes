## 1️⃣ 즉시 발급 가능 — 신청만 하면 당일 연동 (우선 처리)

### 카카오(다음) 검색 API ⭐ 최우선
- **URL**: https://developers.kakao.com → 내 애플리케이션 → 앱 생성 → REST API 키
- **자격**: 카카오 계정만 있으면 즉시. 심사 없음
- **범위**: 다음 웹문서·블로그·카페·동영상 검색 → 네이버가 못 보는 카페/티스토리 영역 커버
- **비용**: 무료 쿼터(일 단위, 검색 계열 수만 건) — 현재 트래픽이면 무료로 충분
- **준비물**: 앱 이름 "DragonEyes", 회사명 드래곤아이즈 주식회사
- **연동 후**: 환경변수 `KAKAO_REST_API_KEY` Railway 등록 → 네이버 분석 탭과 동일 패턴으로 코드 추가 (키 받으면 바로 작업 가능)

### ~~Google Custom Search (CSE) JSON API~~ → ❌ 신규 가입 불가 (2026-07-06 확인)
- 구글 공식 공지: **Custom Search JSON API는 신규 고객 마감**, 기존 고객도 2027-01-01까지 전환 요구
- 참고: Bing Search API도 2025-08 은퇴 — 전통적 웹검색 API 시장 자체가 축소됨
- 구글 권장 대체재(Vertex AI Search)는 50개 도메인 한정이라 전체 웹 탐지 용도엔 부적합

### Brave Search API (구글 CSE 대체) ⭐
- **URL**: https://api-dashboard.search.brave.com → 가입 즉시 사용 (심사 없음)
- **범위**: 독립 웹 인덱스 전체 검색 — 해외 호스팅 도박·유해 사이트 커버
- **비용**: 월 $5 무료 크레딧(약 1,000쿼리) + 초과 시 1,000쿼리당 $3~5. **카드 등록 필수·지출 상한 없음** → 코드에서 일일 호출 상한 필수 구현
- **조건**: 프로젝트 웹사이트/소개 페이지에 "Brave Search API" 어트리뷰션 표기 필요
- **연동 후**: `BRAVE_SEARCH_API_KEY` 환경변수

## 2️⃣ 심사 필요 — 신청서 제출 대상

### Meta (인스타그램)
- **경로 A — Hashtag Search API**: Instagram 비즈니스 계정 + Meta 앱 리뷰(App Review) 필요.
  해시태그당 최근 미디어 조회 가능(계정당 7일 내 30개 해시태그 제한)
  - 신청: https://developers.facebook.com → 앱 생성 → Instagram Graph API → App Review 제출
  - 준비물: 사업자등록증(드래곤아이즈 주식회사), 앱 사용 목적 설명 영상(스크린캐스트), 개인정보처리방침 URL
  - 심사 기간: 통상 수 일~수 주
- **경로 B — Meta Content Library**: 공개 콘텐츠 대량 조회(구 CrowdTangle 대체). **학술·비영리 연구기관 우선** — 영리기업은 승인 어려움. 교육청/대학 협력 시 공동 신청 카드로 활용
- **아동안전 명분 팁**: 신청서에 NCMEC·WeProtect 등 협력 프레임워크 명시, "아동 성착취·딥페이크 유통 모니터링 및 신고" 목적 기재

### TikTok
- **Research API**: 비영리·학술 연구자 중심(미국·유럽 우선). 영리기업 직접 신청은 승인율 낮음
  - 신청: https://developers.tiktok.com/products/research-api/
- **현실적 대안**: 키워드 풀에 틱톡 유도 패턴이 이미 있으므로(타 플랫폼에서 틱톡 이동 유도 탐지), 당장은 간접 탐지 유지. 교육청 사업 확정 후 연구기관 명의 공동 신청

### X (구 트위터)
- **API v2 유료 티어**: 무료 티어는 검색 불가. Basic(월 $200 수준)부터 최근 7일 검색 가능, Pro(월 $5,000)는 전체 아카이브
  - 신청: https://developer.x.com → 프로젝트 생성 → 유료 구독
- **판단 필요**: 월 고정비 대비 효과. 운영비 모델(모니터링 1인 $17)에 큰 영향 → 교육청/국회 예산 확보 후 도입 권장

## 3️⃣ 공식 검색 API가 없는 플랫폼 (기술적 대안)

| 플랫폼 | 상황 | 대안 |
|---|---|---|
| 텔레그램 | Bot API로는 공개 채널 검색 불가 | 공개 채널 한정 MTProto 클라이언트(Telethon) 검토 — 약관·법률 검토 선행 필수 |
| 디스코드 | 검색 API 없음. 봇이 초대된 서버만 접근 | 현행(초대 링크 기반 수동 탐색) 유지 |
| 로블록스/마인크래프트 | 게임 내 채팅 API 없음 | 현행(유튜브 경유 간접 탐지) 유지 |
